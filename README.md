# AWS EC2 Cloud 사용한 서버 배포 시험 repository

Used:

- Node.js v20.10.0
- express@4.18.2
- path@0.12.7
- AWS EC2 (Free Tier)
- Tailwindcss
- prettier

서버 배포 시험 프로젝트로, `GET` 요청에 대해서만 구현되어있는 간단한 서버 및 클라이언트 입니다.

http://52.78.227.153:3000/  
http://52.78.227.153:3000/login

현재는 위 두 페이지에 접속 가능합니다 (2023.11.28)
